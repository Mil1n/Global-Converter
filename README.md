# Global Converter — красивый realtime-конвертер валют и единиц

Современный веб-конвертер с акцентом на удобство, визуал и надёжность:
- realtime-курсы валют,
- realtime-дата и время,
- офлайн-режим на последнем сохранённом курсе,
- конвертер популярных единиц измерения.

## Что умеет

- Загружает **актуальные курсы** через `open.er-api.com`.
- При отсутствии интернета автоматически включает **офлайн-резерв** из `localStorage`.
- Поддерживает **все валюты, которые возвращает API** (включая экзотические коды).
- Показывает **время последнего обновления курсов**.
- Отображает **живые дату/время** в интерфейсе (обновление каждую секунду).
- Конвертирует единицы:
  - длина,
  - вес,
  - температура,
  - площадь.

## UX/UI улучшения

- Glassmorphism-карточки, мягкие световые орбы, глубина и акценты.
- Улучшенные состояния фокуса и hover-эффекты для контролов.
- Адаптивная сетка для мобильных и десктопа.
- Более чистая и контрастная визуальная иерархия.

## Исправленные баги

- Исправлена реакция на изменение `select` (используется событие `change`, а не только `input`).
- Добавлена защита от некорректного отрицательного ввода для не-температурных категорий.
- Улучшены сообщения валидации для более понятной обратной связи.

## Быстрый старт

1. Скачайте/клонируйте репозиторий.
2. Откройте `index.html` в браузере.

> Никакой сборки не требуется: это чистый HTML/CSS/JavaScript.

## Файлы

- `index.html` — структура интерфейса.
- `styles.css` — визуал, адаптив, анимации.
- `script.js` — загрузка курсов, офлайн-кэш, конвертация валют/единиц, валидации.
