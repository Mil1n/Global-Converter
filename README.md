## ✨ Возможности

- 💱 **Валюты в реальном времени** через API `open.er-api.com`.
- 🌐 **Офлайн-режим**: при отсутствии сети используются последние сохранённые курсы из `localStorage`.
- 🗺️ **Широкое покрытие валют**: доступны все коды валют, которые возвращает API.
- 🕒 **Живые дата и время**: отображение обновляется каждую секунду.
- 📏 **Конвертер единиц**:
  - длина,
  - вес,
  - температура,
  - площадь.
- 🎨 **Современный интерфейс**: стеклянные карточки, плавные акценты, удобная адаптивная сетка.

## 🧠 Как это работает

### 1) Валютный модуль
- Загружает актуальные курсы с:
  - `https://open.er-api.com/v6/latest/USD`
- Автоматически заполняет селекты валют.
- Поддерживает конвертацию между любыми двумя валютами из полученного списка.

### 2) Офлайн-fallback
- После успешной загрузки курсы кэшируются в `localStorage`:
  - ключ: `currency_rates_cache_v1`
- Если интернет недоступен — приложение переключается на последний доступный кэш.

### 3) Конвертер единиц
- Использует внутренние коэффициенты и отдельную формулу для температуры (`°C`, `°F`, `K`).
- Мгновенно пересчитывает результат при изменении параметров.

- Корректная обработка `select` через событие `change`.
